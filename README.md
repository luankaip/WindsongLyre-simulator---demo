# 风物之琴模拟器(测试)

## BPM

可指定音符默认休息时间

默认BPM: 60

默认延迟: 1000ms

| 数字 | 延迟(ms) |
| :--: | :------: |
|  0   |   4000   |
|  1   |   2000   |
|  2   |   1000   |
|  3   |   500    |
|  4   |   250    |
|  5   |   125    |
|  6   |   62.5   |
|  7   |  31.25   |

## 乐谱格式

+ 字母大写-->升调
+ 字母小写-->平调
+ () -->内部音符同时发音
+ | -->指定前面音符休息时间长短
+ 数字/多个数字相加-->前面音符休息时间长短
